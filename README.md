# 💫 About Me:
I'm a Computer Science student passionate about building secure, intelligent, and scalable software. My interests lie at the intersection of cybersecurity, AI, and modern full-stack development, where I enjoy turning ideas into production-ready applications that solve real-world problems. I've built projects ranging from phishing detection systems and SOC platforms to AI-powered document intelligence, gaining hands-on experience in system design, authentication, cloud services, and debugging production environments. I'm always exploring new technologies, refining my engineering skills, and looking for opportunities to build software that's both impactful and secure.
